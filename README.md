# Rivet

## Running

```shell
# launch server
$ cargo run
```

Access server at http://localhost:8000

If Ctrl+C isn't sufficient to kill the server (this seems to be the case on Windows) visit
[/quit](http://localhost:8000/quit) to kill the server.
